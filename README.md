# 💰 Assistente Virtual Financeiro com IA

Este projeto consiste em um assistente virtual focado em educação financeira, desenvolvido em Python utilizando a API da OpenAI.

O sistema permite interação via terminal, respondendo perguntas sobre finanças, investimentos e juros, além de realizar simulações simples.

---

## 🚀 Tecnologias utilizadas
- Python  
- OpenAI API  

---

## 💡 Funcionalidades
- Responde dúvidas financeiras  
- Explica conceitos como juros compostos  
- Simula cálculos simples  
- Interação via terminal  

---

## 🎯 Objetivo
Aplicar conceitos de inteligência artificial, linguagem natural e experiência do usuário na criação de um assistente virtual financeiro simples e funcional.

---

## ▶️ Como usar

1. Instale as dependências:

pip install openai

2. Configure sua API Key:

**Windows:**   setx OPENAI_API_KEY "sua-chave-aqui"

**Linux/Mac:**  export OPENAI_API_KEY="sua-chave-aqui"

3. Execute o projeto:

python main.py

---

## ⚠️ Observação
A chave da API não está incluída por segurança.
